# 推荐MOOC课程及书籍

## MOOC课程

### 1. Python从入门到放弃

#### Python基础教学（入门）

[编程基础: Python](https://cn.udacity.com/course/programming-foundations-with-python--ud036)

#### 密西根大学专项课程（入门）

[Python for Everybody 专项课程](https://www.coursera.org/specializations/python)

#### 莱斯大学专项课程（用Python做游戏学编程，有一定难度）

[Fundamentals of Computing 专项课程](https://www.coursera.org/specializations/computer-fundamentals)

#### MIT课程（入门到放弃，MIT的都是神级课程，值得入坑）

[Introduction to Computer Science and Programming Using Python](https://www.edx.org/course/introduction-computer-science-mitx-6-00-1x-11)
[Introduction to Computational Thinking and Data Science](https://www.edx.org/course/introduction-computational-thinking-data-mitx-6-00-2x-6)

#### 收费网络课程

1. [Udacity纳米学位：人工智能编程基础](https://cn.udacity.com/course/ai-programming-foundations-nanodegree--nd089-cn)
2. [Udacity纳米学位：无人驾驶入门](https://cn.udacity.com/course/intro-to-self-driving-cars--nd113-cn)
3. [Udacity纳米学位：无人驾驶工程师](https://cn.udacity.com/course/self-driving-car-engineer--nd013)



### 2. Java从入门到放弃

#### 北大课程（入门到放弃）

[Java程序设计](https://www.coursera.org/learn/java-chengxu-sheji)

#### 圣地亚哥与杜克联合出品课程（入门到放弃）

[Object Oriented Programming in Java 专项课程](https://www.coursera.org/specializations/object-oriented-programming)

#### 普林斯顿大学课程(顶级算法课程，有意思，但难度很大)

[Algorithms, Part I](https://www.coursera.org/learn/algorithms-part1)
[Algorithms, Part II](https://www.coursera.org/learn/algorithms-part2)



### 3. 前端3剑客从入门到放弃

##### 密西根大学课程（入门）

[Web Design for Everybody (Basics of Web Development and Coding) 专项课程](https://www.coursera.org/specializations/web-design)

##### 香港科技大学（入门到放弃，主要学习Angular框架及相关内容）

[Full Stack Web and Multiplatform Mobile App Development 专项课程](https://www.coursera.org/specializations/full-stack-mobile-app-development)
